# Smart Waste Management System

## Overview
AI-powered platform to monitor waste bins, predict fill levels, and optimize collection routes.

## Features
- Real-time bin monitoring
- Fill level prediction (ML)
- Route optimization
- Interactive map dashboard

## Tech Stack
- Frontend: React + Tailwind
- Backend: FastAPI
- ML: Random Forest / LSTM
- Maps: Leaflet

## Installation
```bash
npm install
npm run dev
